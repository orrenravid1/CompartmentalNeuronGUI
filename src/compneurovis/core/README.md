# Core Package

`compneurovis.core` defines the stable data and view model:

- `Field`
- `MorphologyGeometry`
- `GridGeometry`
- `Document`
- `LayoutSpec`
- `View3DHostSpec`
- `ViewSpec`
- `ControlSpec`
- declarative binding helpers such as `AttributeRef` and `SeriesSpec`
