# Field Model

Use `Field` for measured, simulated, or replayed dense data.

Examples:

- membrane voltage by segment by time
- current source density by channel by time
- surface height by x by y

Treat “trace”, “surface”, and “heatmap” as views over a field rather than new foundational data types.
