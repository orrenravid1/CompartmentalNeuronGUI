# scratch/

One-off exploratory scripts, prototypes, and technical spikes. Nothing here is maintained,
tested, or indexed.

**Use this folder when you want to:**
- Explore a new library or API before deciding whether to integrate it
- Run a quick experiment that doesn't belong in the polished `examples/`
- Write a throwaway script to validate an assumption

**Do not use this folder for:**
- Reference implementations or demos meant for users → use `examples/`
- Reusable utilities → add them to the package under `src/compneurovis/`
- Tests → use `tests/`

Files here are not picked up by `scripts/generate_indexes.py` and will not appear
in the docs site. They may be deleted or broken at any time.
